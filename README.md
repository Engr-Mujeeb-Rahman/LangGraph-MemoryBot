# LangGraph MemoryBot

A **GPT-style chatbot with persistent memory**, built using **LangGraph** and **Streamlit**.  
This chatbot stores all conversations in a **SQLite database**, allowing users to resume chats anytime.  
It uses **Google Gemini LLM** for AI responses and supports multiple chat threads.

---

## Features

- Persistent chat memory (resumes after restart)  
- Context-aware responses using Google Gemini LLM  
- Multiple chat threads  
- Clean, modern **Streamlit UI** with dark theme  
- Easily deployable and customizable
